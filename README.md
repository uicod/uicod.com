# uicod.com
This website was built with HTML5, CSS3 and Parcel.

## Usage
#### Clone the repository
`git clone https://github.com/uicod/uicod.com.git`

#### Into the repo directory
`cd uicod.com`

#### Install NPM dependicies
`npm install`

#### Start the project
`npm run start`

#### Bundle the project
`npm run build`
